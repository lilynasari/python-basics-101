  学习心得：  
    1.将项目不同功能部分可以进行拆分，在不同函数下完成不同功能，在通过调用函数的方式将其连接在一起  
    2.尽量不要定义作用与全局范围的变量，让变量都仅为函数内部变量  
    3.输入语句的位置要选好避免重复输入（如利用函数调用）  
    4.写代码前先分析好要做什么事情，选好循环语句避免if else深层嵌套这类情况发生
